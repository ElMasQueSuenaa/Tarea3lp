Extraer el tar.gz y se quedará una carpeta llamada "Tarea3lp".
Quedarán todos los archivos dentro de la carpeta ya mencionada.
Entras a la carpeta y con shift+click derecho abres una shell de linux.
Ejecutas el comando "make" y posteriormente "make run" y podrás jugar "No Java`s Sky".
Una vez que se haya terminado de jugar escribe "make clean" para que se borren los archivos .bin
